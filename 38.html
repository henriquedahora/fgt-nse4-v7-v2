A network administrator wants to set up redundant IPsec VPN tunnels on FortiGate by using two IPsec VPN tunnels and static routes.

* All traffic must be routed through the primary tunnel when both tunnels are up
* The secondary tunnel must be used only if the primary tunnel goes down
* In addition, FortiGate should be able to detect a dead tunnel to speed up tunnel failover

Which two key configuration changes are needed on FortiGate to meet the design requirements? (Choose two,)

A. Configure a high distance on the static route for the primary tunnel, and a lower distance on the static route for the secondary tunnel.

B. Enable Dead Peer Detection.

C. Configure a lower distance on the static route for the primary tunnel, and a higher distance on the static route for the secondary tunnel.

D. Enable Auto-negotiate and Autokey Keep Alive on the phase 2 configuration of both tunnels.


Um administrador de rede deseja configurar túneis VPN IPsec redundantes no FortiGate usando dois túneis VPN IPsec e rotas estáticas.

* Todo o tráfego deve ser roteado pelo túnel primário quando ambos os túneis estiverem ativos
* O túnel secundário deve ser usado somente se o túnel primário ficar inativo
* Além disso, o FortiGate deve ser capaz de detectar um túnel morto para acelerar o failover do túnel

Quais são as duas principais mudanças de configuração necessárias no FortiGate para atender aos requisitos de design? (Escolha dois,)

A. Configure uma distância alta na rota estática para o túnel primário e uma distância menor na rota estática para o túnel secundário.

B. Habilitar Detecção de Dead Peer.

C. Configure uma distância menor na rota estática para o túnel primário e uma distância maior na rota estática para o túnel secundário.

D. Habilite Auto-negociate e Autokey Keep Alive na configuração da fase 2 de ambos os túneis.

Resposta(s):

B, C

Explicação:

B - porque o cliente exige que os túneis notifiquem quando um túnel for desativado. O DPD é projetado para esse propósito. Para enviar um pacote por um firewall para determinar um failover para o próximo túnel após um determinado período de tempo sem receber uma resposta de seu par.
C - lembre-se na hora de escolher uma rota em relação à Distância Administrativa. A rota com a menor distância para essa rota específica será escolhida. Assim, ao configurar uma distância de roteamento menor no túnel primário, significa que o túnel primário será escolhido para rotear os pacotes em direção ao seu destino.
